Changelog

Version 0.1.0 - Beta

Release Date: 10/10/2019

  • Add alt text to some images that were missing it.
  • Implemented the sorting to select box options.
  • Fixed an issue with modal appearing behind the navigation bar
  • Fixed a bug with the time picker on the search page.
  • Caught a few spelling mistakes that were on the loose.
  • Tidied up some comments hidden in the code.
This is the final release before the project is submitted for grading. This site is still in beta as there are many things that need to be fixed before it can be considered stable. A list of known issues can be found here, anything other bugs please remember to contact me using the form above. Thank you to everyone who helped test out the development of this project and I hope you enjoy using Cookbook!

Version 0.0.5 - Alpha

Release Date: 05/10/2019

  • A major cleanup of the code has been made in the background, useless bits have been removed, comments are more descriptive, Marie Kondo would be proud.
  • The first draft of the report has been written, you can find it at the Github link in the footer if you're interested.
  • Fixed a bug where the like button was not blue on the home page for liked recipes.
  • Fixed a bug with the with the stars not rendering correctly in the rating modal.

Version 0.0.4 - Alpha

Release Date: 01/09/2019

  • Add 404 popup on this page in case something breaks.
  • Fix a bug with the rating stars rendering on mobile devices, if you ever see boxes instead of stars please let me know.
  • The search bar now works! When testing, the advanced search does nothing yet but you should be able to do simple text searches for text in recipes.
  • Some minor tweaks have been made to the layout of some parts.

Version 0.0.3 - Alpha

Release Date: 21/08/2019

  • Fix bug with comments on the recipe cards on the home page. Now when clicked only the relevant comment section should open, not all of them.
  • Stopped users being allowed to rate the same recipe multiple time.
Some things in this update may make the website slow to load, especially the home page. This problem may get worse as the site grows but if you notice that any part of the website is unreasonably slow please let me know.

Version 0.0.2 - Alpha

Release Date: 20/08/2019

  • Migrated the Database to the live version, no more test recipes this is real folks.
  • Added some "Coming Soon" popup boxes on things that definitely have not been finished. If something doesn't work as you think it's supposed to and you don't get a popup then please let me know.
  • Fixed some issues when using multiple choice inputs where it would be difficult to remove entered values.
  • Hooked up the email form so now emails actually get sent to me!
  • Fixed some small backend issues.

Version 0.0.1 - Alpha

Release Date: 08/08/2019

This is the initial alpha release so everything you see is new but to explain what is included:
  • Home page of submitted recipes in reverse chronological order.
  • User login so that you can submit your own recipes and interact with others.
  • User profile listing submitted recipes, comments and liked recipes.
  • Submission page so that you can submit your own recipes to the site.
  • Recipe page showcasing recipes with ingredient lists, instructions and any comments other users have made.
  • This page with comment form and changelog.
There are numerous features and details that I have not included here but you can check out the sites development over on the Github page that is at the "About this project" link found in the footer below.
